1. Introduction to C++

C++ is a general-purpose programming language created by Bjarne Stroustrup in 1985 as an extension of the C language. It introduces object-oriented programming while retaining C's low-level power, making it ideal for system-level development, game engines, finance systems, and more.

1.1 History

C++ evolved from C with Classes and was standardized by ISO in 1998. Since then, modern versions like C++11, C++14, C++17, and C++20 introduced major improvements such as smart pointers, concurrency, ranges, and modules.

1.2 Setting up

To run C++, install a compiler such as g++ or clang++. For Windows, MinGW or MSVC can be used. For macOS/Linux, g++ is typically pre-installed.

2. Syntax & Basics

The foundation of C++ programming includes variables, data types, control structures, and functions.

#include <iostream>
using namespace std;

int main() {
    cout << "Hello, C++ World!" << endl;
    int x = 42;
    if (x > 10) {
        cout << "x is greater than 10" << endl;
    }
    return 0;
}

Key features include type safety, manual memory management, and compile-time efficiency.

3. Object-Oriented Programming

C++ introduced OOP to C, enabling encapsulation, inheritance, and polymorphism.

class Animal {
public:
    virtual void speak() { cout << "Generic animal sound" << endl; }
};

class Dog : public Animal {
public:
    void speak() override { cout << "Woof!" << endl; }
};

int main() {
    Animal* a = new Dog();
    a->speak(); // Woof!
    delete a;
    return 0;
}

This enables runtime polymorphism and supports large-scale applications.

4. Standard Template Library (STL)

The STL provides containers, algorithms, and iterators, allowing fast and efficient coding.

#include <vector>
#include <algorithm>
#include <iostream>
using namespace std;

int main() {
    vector<int> nums = {5, 2, 9, 1};
    sort(nums.begin(), nums.end());
    for (int n : nums) cout << n << " ";
    return 0;
}

Popular containers include vector, map, set, and unordered_map.

5. Advanced C++

Modern C++ includes powerful features like:

  • Smart Pointers – automated memory management
  • Lambda Expressions – inline anonymous functions
  • Multithreading – parallel execution with <thread>
  • Move Semantics – efficient resource handling
  • Templates & Metaprogramming
#include <memory>
#include <iostream>
using namespace std;

int main() {
    unique_ptr<int> p = make_unique<int>(42);
    cout << *p << endl; // 42
    return 0;
}

6. Conclusion

C++ remains one of the most resilient and high-performance programming languages in the world. It balances low-level control with high-level abstractions, making it the backbone of industries like gaming, finance, embedded systems, and operating systems.
